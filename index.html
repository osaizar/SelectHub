¿Quienes Somos?

En este proyecto proponemos un sistema de selección de software para empresas. Este sistema ayudará a comparar distintos softwares para varios tipos de necesidades, analizando los puntos positivos y negativos de cada uno ayudando así a elegir el software que más se ajuste a las necesidades de la empresa.
